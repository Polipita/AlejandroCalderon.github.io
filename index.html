Valorización de Residuos - MINAM

Este proyecto consiste en el análisis de datos abiertos del Ministerio del Ambiente del Perú (MINAM) sobre la valorización de residuos sólidos a nivel distrital. Se elaboró un dashboard interactivo en Power BI que permite visualizar y comparar los tipos y volúmenes de residuos orgánicos e inorgánicos valorizados por distrito y departamento.

  • Análisis de Datos
  • Power BI
  • Excel

Scraping de Productos

Proyecto en desarrollo que utiliza Python y la librería BeautifulSoup para extraer precios y características de productos de diferentes páginas web de retail en Perú. El objetivo es crear una base de datos actualizada que permita comparar precios y realizar análisis de competencia. Planeo integrarlo con Power BI para visualizar los resultados.

  • Python
  • Web Scraping
  • BeautifulSoup
